Static Positioning

This element is positioned using static positioning, which is the default. It follows the normal document flow.

Relative Positioning

This element is positioned using relative positioning. It's offset by 20px from its normal position.

Absolute Positioning

This element is positioned using absolute positioning. It is 50px from the top and 20px from the left of its nearest positioned ancestor (relative container).

Fixed Positioning

This element uses fixed positioning. It stays in the same place even when you scroll the page.

I stay fixed here!

Sticky Positioning

This element uses sticky positioning. It sticks to the top when you scroll past it and stays fixed until the section ends.
